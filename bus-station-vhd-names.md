# Bus Station Themed VHD Names

## Current VHDs
- **omnibus** - The main transport hub
- **bus-station** - Central terminal
- **buggy** - Development/testing environment

## Additional Bus Station Themed Names

### Infrastructure
- **terminal** - Main departure/arrival point
- **depot** - Vehicle storage and maintenance
- **garage** - Repair and service center
- **platform** - Loading/unloading area
- **concourse** - Passenger waiting area

### Operations
- **dispatch** - Route coordination
- **schedule** - Timing and routing
- **fleet** - Vehicle collection
- **route** - Path management
- **transfer** - Connection point

### Services
- **ticket** - Access control
- **lounge** - Premium waiting area
- **cargo** - Freight handling
- **shuttle** - Connector service
- **express** - Fast transport

### Technical
- **hub** - Network central point
- **gateway** - Entry/exit point
- **junction** - Intersection point
- **transit** - Movement system
- **mobile** - Portable service

### Recommended for NB-Ecosystem
- **terminal** - For the main ecosystem VHD
- **hub** - If this is your central development environment
- **transit** - For moving projects between environments
- **gateway** - If this serves as an entry point to other systems
